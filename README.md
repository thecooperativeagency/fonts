# Fonts — The Cooperative Agency

CDN-hosted fonts for use in emails, web pages, and SEO content.

## BMW Type Next Pro

**CSS import (all weights):**
```html
<link rel="stylesheet" href="https://thecooperativeagency.github.io/fonts/bmw.css"/>
```

**Usage:**
```css
font-family: 'BMWTypeNext', Arial, sans-serif;
font-weight: 400; /* Regular */
font-weight: 700; /* Bold */
```
